## Como rodar a aplicação

Para o download do projeto, siga as instruções abaixo:

```
1. git clone https://github.com/mateusgs29/api-tarefas.git
2. cd api-tarefas
```

Instale as dependências e inicie o projeto:

```
1. npm install
2. npm start
```
